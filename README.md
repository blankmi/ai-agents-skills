# AI Agent Skills

A small collection of skills converted for use with different AI agents.

## What's in here

Right now this repo only contains a **[playwright-cli](https://github.com/microsoft/playwright-cli)** skill that's been converted from [Claude Code](https://docs.anthropic.com/en/docs/claude-code) to work with Gemini CLI and Codex. The original skill is built for Claude Code — these are ports of it.

## Repository Structure

- `gemini-skills/` — Skills for [Gemini CLI](https://github.com/google-gemini/gemini-cli)
- `codex-skills/` — Skills for [Codex](https://github.com/openai/codex)

## Getting Started

Check the README in the folder for the agent you're using:
- [Gemini Skills README](./gemini-skills/README.md)
- [Codex Skills README](./codex-skills/README.md)

## A note on maintenance

I might add more skills here if I need one and it's not available for an AI agent I use, but I'm not actively maintaining or expanding this repo. Use what's here as-is, or feel free to fork and build on it.
